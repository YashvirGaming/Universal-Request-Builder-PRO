<h2>🚀 Overview</h2>

<p>
Universal Request Builder PRO is a premium desktop tool that allows users to visually build advanced HTTP request flows and generate fast standalone Python CLI checkers.
</p>

<p>
Perfect for automation, API testing, chained request workflows, token capture flows, combo processing, and proxy-powered request systems.
</p>

<hr>

<h2>✨ Features</h2>

<h3>🧩 Multi Flow Request Engine</h3>

<ul>
<li>Unlimited request steps</li>
<li>Reorder request chain</li>
<li>Duplicate requests</li>
<li>GET / POST / PUT / PATCH / DELETE / HEAD</li>
</ul>

<h3>🎯 Response Logic</h3>

<ul>
<li>Success keywords</li>
<li>Fail keywords</li>
<li>Retry detection</li>
<li>Custom response matches</li>
<li>Status code support</li>
</ul>

<h3>📦 Captures & Variables</h3>

<ul>
<li>Capture JSON keys</li>
<li>Nested captures</li>
<li>Reuse captures in next requests</li>
</ul>

<pre>
Authorization: Bearer {token}
https://site.com/user/{user.id}
</pre>

<h3>🌐 Proxy Support</h3>

<p>Supports raw proxy formats:</p>

<pre>
ip:port
ip:port:user:pass
http://user:pass@ip:port
</pre>

<p>Automatically converted for HTTPX.</p>
